## 摘要
脑电图（EEG）是非侵入式脑机接口（BCI）的基石，但传统的解码依赖于碎片化、任务特定的架构，严重限制了跨任务的可扩展性。尽管在大规模语料库上预训练的脑电图基础模型有望实现通用脑解码，但当前的后训练依赖于任务隔离的微调。这种静态范式限制了跨异质任务的知识迁移，阻碍了模型的可扩展性，并带来了与任务数量线性增长的计算和存储开销。为克服这些瓶颈，我们将下游适应建模为跨任务持续学习问题，并提出了EvoBrain，一种用于统一脑电图解码的动态、任务感知的持续学习框架。EvoBrain通过两个互补组件解决可塑性-稳定性权衡：（1）神经频谱任务归一化（NSN）将新任务与历史统计信息对齐，同时重新校准频谱响应以处理分布和神经频谱偏移；（2）响应亲和蒸馏（RAD）结合时间依赖重放，保留旧任务的响应几何结构，并促进频谱兼容任务之间的选择性知识迁移，有效缓解遗忘。在六个不同BCI任务上的广泛评估表明，EvoBrain在不同基础骨干网络上持续超越最先进方法，最优地平衡了可塑性和稳定性。据我们所知，这项工作开创了脑电图领域的跨任务持续学习，推动了统一、一劳永逸的脑解码系统的实现。

## Abstract
Electroencephalography (EEG) is the cornerstone of non-invasive brain-computer interfaces (BCIs), yet conventional decoding relies on fragmented, task-specific architectures that severely limit cross-task scalability. While EEG foundation models pre-trained on massive corpora promise universal brain decoding, current post-training depends on task-isolated fine-tuning. This static paradigm restricts knowledge transfer across heterogeneous tasks, hinders model scalability, and incurs computational and storage overheads that scale linearly with task count. To overcome these bottlenecks, we formulate downstream adaptation as a cross-task continual learning problem and propose EvoBrain, a dynamic, task-aware continual learning framework for unified EEG decoding. EvoBrain addresses the plasticity-stability trade-off via two complementary components: (1) Neuro-Spectral Task Normalization (NSN) aligns incoming tasks with historical statistics while recalibrating spectral responses to handle distributional and neuro-spectral shifts; and (2) Response-Affinity Distillation (RAD), combined with time-dependent replay, preserves old-task response geometry and promotes selective knowledge transfer between spectrally compatible tasks, effectively mitigating forgetting. Extensive evaluations across six distinct BCI tasks demonstrate that EvoBrain consistently surpasses state-of-the-art methods across diverse foundation backbones, optimally balancing plasticity and stability. To our knowledge, this work pioneers cross-task continual learning in the EEG domain, advancing the realization of a unified, one-for-all brain decoding system.

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：脑电图（EEG）是非侵入式脑机接口（BCI）的主要模态，凭借高时间分辨率在情感识别、运动想象、睡眠分期等任务中广泛应用。传统方法依赖任务特定的架构，限制了跨任务泛化。近年来，EEG基础模型（如CBraMod、EEGMamba、CSBrain、DeeperBrain）通过大规模无监督预训练学习通用表征，有望实现“一模型解决所有任务”的通用脑解码。
- **核心问题**：当前的后训练（post-training）阶段采用任务隔离的静态微调范式，即每个下游任务独立微调整个模型。这导致两个根本性瓶颈：
  1. **知识迁移受限**：模型无法在异质BCI任务之间传递和整合知识，阻碍了高层次、不变神经模式的发现；
  2. **资源开销线性增长**：每增加一个新任务都需要从预训练权重重新开始训练，并存储完整副本，计算和存储成本与任务数量成正比。
- **整体含义**：迫切需要一种动态、任务感知的后训练策略，使单个统一的基础模型能够增量地适应不同任务序列，同时不遗忘旧任务的知识。该问题自然形式化为跨任务持续学习（Continual Learning, CL）问题，需平衡学习可塑性（对新任务的适应能力）与记忆稳定性（对旧任务的保持能力）。

## 2. 方法论：核心思想、关键技术细节

### 2.1 整体框架：EvoBrain

- **核心思想**：将EEG基础模型的后适应建模为跨任务持续学习问题。模型从预训练状态 \(M_0\) 开始，顺序接收异质EEG任务流 \(\{T_1, T_2, \dots, T_N\}\)，每个任务附带轻量级分类头。EvoBrain通过两个互补模块解决可塑性-稳定性权衡：
  1. **神经频谱任务归一化（NSN）**：增强对新任务的学习可塑性；
  2. **响应亲和蒸馏（RAD）**：结合时间依赖重放，维护旧任务的记忆稳定性。

- **算法流程**（见Algorithm 1）：
  - 对于第一个任务，仅使用交叉熵损失 \(L_{CE}\) 优化；
  - 对于后续任务 \(T_i (i>1)\)：
    - 根据时间依赖策略（公式4）从存储的历史任务中采样重放批次；
    - 对当前批次数据应用NSN归一化；
    - 联合优化分类损失和RAD损失：\(L = \lambda_{ce} L_{CE} + (1-\lambda_{ce}) L_{RAD}\)。

### 2.2 神经频谱任务归一化（NSN）

- **组成**：原型引导时间归一化（PTN）+ 任务条件频谱残差调制（SRM）
- **原型引导时间归一化（PTN）**：
  - 为每个任务维护一个历史原型向量 \(p_i\)（指数移动平均任务均值）；
  - 通过可学习的门控参数 \(\eta^\mu_i, \eta^\sigma_i\) 动态融合当前任务统计量 \((\mu_i, \sigma_i)\) 和历史参考统计量 \((\mu_g, \sigma_g)\)，得到目标矩 \((\mu^\text{tar}_i, \sigma^\text{tar}_i)\)；
  - 计算仿射变换参数 \(\gamma_i, \beta_i\) 并归一化输入 \(X_{tn} = \gamma_i X + \beta_i\)。
- **任务条件频谱残差调制（SRM）**：
  - 对归一化表示 \(X_{tn}\) 沿时间维度做实值傅里叶变换 \(\hat{X}(f) = \text{rFFT}(X_{tn})\)；
  - 构建标准EEG频带（δ, θ, α, σ, β, γ）的软掩码 \(M_b(f)\)，并归一化为 \(W_b(f)\)；
  - 为每个频带分配可学习参数 \(a_b\)，通过 \(\tanh\) 得到门控值 \(g_b \in [-1,1]\)，投影为频谱调制函数 \(G(f)\)；
  - 计算残差频谱 \(\Delta \hat{X}(f) = G(f) \odot \hat{X}(f)\)，经逆傅里叶变换得 \(\Delta X\)；
  - 最终输出 \(X_{nsn} = X_{tn} + \alpha_m \Delta X\)（\(\alpha_m\) 为调制强度）。
- **作用**：PTN减少跨任务分布偏移；SRM自适应增强或抑制任务相关的频谱响应，并产生紧凑的频带门控向量 \(g\) 用于后续亲和分析。

### 2.3 响应亲和蒸馏（RAD）

- **组成**：响应几何蒸馏（RGD）+ 频谱亲和对齐（SAA）
- **时间依赖重放**：对旧任务按公式4计算重放概率（年龄越大、深度越大，概率越高），每个当前小批量搭配一个重放小批量。
- **响应几何蒸馏（RGD）**：
  - 将重放样本输入当前学生模型和冻结的旧任务教师模型（任务 \(T_j\) 训练后保存的检查点 \(M_j\)）；
  - 提取第 \(\ell\) 层编码器的中间响应，经全局平均池化后 \(\ell_2\) 归一化；
  - 计算学生和教师的样本间关系矩阵 \(R_{s,r}^{(\ell)}\) 和 \(R_{t,r}^{(\ell)}\)；
  - 最小化非对角线元素（样本间关系）的均方误差：\(L_{RGD} = \frac{1}{L}\sum_{\ell=1}^L \frac{1}{B(B-1)}\sum_{i\neq j}(R_{s,r,ij}^{(\ell)} - R_{t,r,ij}^{(\ell)})^2\)。
- **频谱亲和对齐（SAA）**：
  - 计算当前任务学生响应 \(H_{s,c}^{(\ell)}\) 和重放教师响应 \(H_{t,r}^{(\ell)}\) 的统计量 \((\mu_c, \sigma_c, \Sigma_c)\) 与 \((\mu_r, \sigma_r, \Sigma_r)\)；
  - 计算统计对齐损失 \(L_{align} = \frac{1}{L}\sum_{\ell} [\|\mu_c-\mu_r\|_2^2 + \|\sigma_c-\sigma_r\|_2^2 + \rho\|\Sigma_c-\Sigma_r\|_F^2]\)；
  - 计算当前任务与重放任务的频带门控向量 \(g_c, g_r\) 之间的符号亲和度 \(\alpha_s\)（公式24-26）；
  - 仅当正负频带门控均相似时激活对齐：\(L_{SAA} = \alpha_s \cdot L_{align}\)，避免无关范式间的负迁移。
- **总RAD损失**：\(L_{RAD} = \lambda_r L_{RGD} + (1-\lambda_r) L_{SAA}\)。

### 2.4 总体损失函数

- 对于第 \(i\) 个任务（\(i>1\)）：\(L = \lambda_{ce} L_{CE} + (1-\lambda_{ce}) L_{RAD}\)；
- 对于第一个任务：\(L = L_{CE}\)。

## 6. 主要结论与发现

1. **EvoBrain有效实现跨任务持续学习**：在六种异质BCI任务上，单骨干模型通过EvoBrain进行增量适应后，平均B-ACC达到单任务微调的91.3%，RFR保持在0.1%~3.9%的低水平，显著优于所有对比基线。
2. **NSN和RAD互补**：NSN提供任务感知的统计对齐和频谱重新校准，RAD通过关系蒸馏和亲和对齐维持稳定并促进选择性转移，两者共同优化可塑性-稳定性权衡。
3. **跨架构兼容**：EvoBrain在Transformer（CBraMod、CSBrain）和Mamba（EEGMamba、DeeperBrain）上表现一致稳定，表明框架不依赖特定架构。
4. **跨顺序鲁棒**：三种不同任务到达顺序下，最终平均B-ACC波动很小（±0.012以内），RFR保持低水平。
5. **正迁移效应**：某些任务的最终性能反而高于即时性能，说明EvoBrain能够从后续任务中受益，实现正向知识迁移（如运动想象任务间的互促）。
6. **频谱调制可解释**：SRM学习到的频谱调制模式与各任务已知的神经生理学先验一致（如睡眠分期增强δ波段、运动想象增强β波段），增强了模型的可解释性。

- **方法设计的针对性与完整性**：
  - NSN结合时间归一化和频谱残差调制，同时处理分布偏移和任务特异频谱响应，是面向EEG信号特性的定制化归一化。

  - 时间依赖重放策略适应EEG任务序列中早期任务更容易遗忘的特点。
- **实验充分性**：在4种异构骨干网络、6个多样任务（覆盖睡眠、运动想象、情感、言语、精神疾病）、3种任务顺序、8种强基线方法下进行验证，消融实验全面，可视化分析深入（表示演化、频谱调制、关系保持）。
- **即插即用性**：EvoBrain可集成到现有主流EEG基础模型，无需修改骨干架构。
- **可解释性**：频谱调制模式与神经生理学频带意义对应，关系保持可视化直观展示了RAD的效果。

## 8. 不足与局限

- **实验限制**：
  - 目前采用任务增量协议（已知任务边界、固定任务特定分类头），未考虑更灵活的场景，如任务身份模糊、延迟标签、动态头选择、统一输出空间。
  - 仅使用固定记忆预算的重放策略，未探讨隐私保护（联邦学习、原型重放、生成重放）等替代方案。
  - 未在更大规模基础模型（如更大的BERT-like EEG模型）上验证，也未评估模型参数规模对持续学习的影响。
- **计算资源开销**：
  - 虽然避免了为每个任务独立训练完整模型，但重放和蒸馏仍需要额外计算（每个迭代多处理一批重放数据）。
  - 未报告训练总时长、单任务增量时间与推理延迟，难以评估实际部署效率。
- **潜在偏差风险**：
  - 任务顺序的选择虽测试了三组，但可能仍无法完全覆盖真实应用中所有可能的到达模式。
  - 数据集均经过标准化预处理，与真实在线连续流场景存在差距（如信号长度变化、在线校准需求）。
- **应用限制**：
  - 当前框架需要为每个新任务添加分类头，当任务数量极大时，分类头存储仍可能成问题。
  - 频谱调制中使用的固定频带划分（δ/θ/α/σ/β/γ）可能不适用于所有罕见范式（如高频成分）。
  - 未考虑多模态脑成像数据（如fMRI、MEG）的扩展。
- **局限性总结**：论文坦诚指出，未来工作应关注更开放的在线协议、隐私友好的记忆机制，以及更深的神经生理学验证。
