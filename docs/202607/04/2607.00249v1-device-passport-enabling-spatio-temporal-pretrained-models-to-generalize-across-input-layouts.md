## 摘要
新设备布局由于缺乏针对每种特定布局的大型数据集而构成一个具有挑战性的建模问题。如果生物信号基础模型能够有效泛化到新布局，它们将提供一种可行的解决方案。为了改进跨布局迁移，我们研究了当预训练布局与下游解码布局显著不同时，不同通道嵌入技术的表现。我们提出了设备护照，这是一种新的通道嵌入技术，它学习专家和混合模型，以每个通道的功能活动和元数据作为输入。这与先前的嵌入方法形成对比，后者通常仅使用功能信息或仅使用元数据来查找学习或固定的位置嵌入。在受控的子集迁移实验和向耳部脑电图的现实迁移中，设备护照整体上具有竞争力，并且在推动这项工作的布局迁移方案中，相比最强的学习基线有所改进。这些结果表明，在新设备上重用大规模预训练生物信号模型时，通道嵌入设计是一个关键考虑因素。

## Abstract
New device layouts pose a challenging modeling problem due to the lack of large datasets for each specific layout. Biosignal foundation models offer a plausible solution if they are able to generalize to new layouts effectively. To improve cross-layout transfer, we study how different channel embedding techniques behave when pretraining layouts differ substantially from the downstream decoding layout. We propose Device Passport, a new channel embedding technique that learns experts and mixture models that take each channel's functional activity and metadata as input. This contrasts with prior embedding methods, which typically use only functional information or only metadata to look up learned or fixed positional embeddings. Across controlled subset-transfer experiments and realistic transfer to ear-EEG, Device Passport is competitive overall and improves over the strongest learned baseline in the layout-transfer regimes that motivate this work. These results suggest that channel embedding design is a key consideration when reusing large-scale pretrained biosignal models on new devices.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将通道嵌入视为一个**专家混合**问题，在预训练阶段学习一组可共享的专家嵌入，下游时通过轻量适配器（基于元数据+功能活动）为每个新通道分配专家权重，生成其嵌入。
- **关键技术细节**：
  - **输入**：对每个通道 \( c \)，从非掩码的patch嵌入中池化出活动摘要 \( z_c \)，并与元数据 \( m_c \)（XYZ电极坐标）拼接。
  - **专家嵌入**：预训练时学习固定数量的专家嵌入 \( \{E_k\} \)（论文默认为10个）。
  - **混合模型变体**：
    - **MLP Experts**：用MLP将 \( [z_c, m_c] \) 映射为专家权重 \( w_c = \text{softmax}(\text{MLP}([z_c, m_c])) \)，通道嵌入为 \( e_c = \sum_k w_{c,k} E_k \)。
    - **Cross-Attention Experts**：将 \( [z_c, m_c] \) 作为查询，专家嵌入作为键/值，通过交叉注意力生成通道嵌入。
  - **冻结策略**：下游微调时，专家嵌入冻结，仅更新混合模块（MLP或Cross-Attention适配器），避免重学整个嵌入空间。
- **对比先前方法**：传统方法仅用元数据（如XYZ正弦编码）或仅用功能活动（如ACPE），而Device Passport二者兼用，且通过专家共享实现跨布局的隐式正则。

## 6. 主要结论与发现

1. **布局不匹配显著降低性能**：预训练布局与下游布局差异越大，AUROC下降越明显（如额叶→标准 vs 标准→标准）。
2. **ACPE在极低数据（<20 subjects）下表现最好**，因其可训练参数少、易于快速适应。
3. **在中等样本量（20–200 subjects）及布局迁移情况下，Device Passport优于ACPE**，说明元数据+功能活动联合嵌入在积累少量数据后能更有效利用预训练专家知识。
4. **在真实耳-EEG迁移中**：MLP Experts在9个LOSO受试者中7个优于ACPE，2个稍差（对应已知电极接触不良的受试者）。
5. **对噪声通道敏感**：当电极接触差导致功能活动摘要 \( z_c \) 失真时，混合模型可能分配错误专家权重，是当前主要局限。

## 7. 优点

1. **问题视角新颖**：明确将通道嵌入定位为跨布局迁移的核心瓶颈，而非简单当作预处理步骤。
2. **方法设计巧妙**：将通道嵌入建模为专家混合，预训练共享专家+下游轻量适配器，符合迁移学习“预训练-微调”范式。
3. **实验控制严谨**：保持骨干、预训练/微调设置完全一致，仅改变嵌入方式；采用多随机种子消除偶然性。
4. **结果具有实际意义**：在数据稀少的真实设备场景（耳-EEG）中取得提升，证明了方法的驱动场景吻合度。

## 8. 不足与局限

1. **对噪声通道脆弱**：依赖功能活动摘要，电极接触不良时性能下降（EESM17中两个受试者表现差）。
2. **依赖可用元数据**：需要精确的电极3D坐标或等效位置信息，某些设备可能无法提供。
3. **验证范围有限**：仅在一类骨干（CBraMod）和两个EEG数据集上测试，未推广到EMG、EKG或其他架构，也未探索不同专家数量、不同混合模型复杂度的影响。
4. **缺少消融实验**：未单独评估元数据或功能活动各自的贡献，也未对比仅用坐标的基线（如XYZ）与Device Passport的增益分解。
