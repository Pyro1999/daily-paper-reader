## 摘要
尽管非言语行为和表达性动作对于自然的人机交互至关重要，但现有方法往往忽略了一个关键要素：人类的内部认知状态。通常，主动的多智能体系统会在不恰当的时刻打断人类，导致认知过载和任务绩效下降。本文提出了一种生成“认知对齐”的多智能体交互框架，增强了机器人系统在高心智工作负荷和高度投入时，能够根据情境将通信推迟到智能体系统用户的能力。我们介绍了一种闭环架构的设计与实现，该架构探索了自主任务执行与实时神经生理焦点之间的相互作用。使用消费级脑机接口（BCI），我们的方法在人类执行投入诱导任务时，持续监测脑电图（EEG）频谱带功率。我们提出了一种由投入驱动的流水线，其中基于HTTP的信令机制在检测到高投入时，将主智能体的感官输入和音频输出置于保持状态。这使得辅助智能体能够在后台无缝处理复杂的委托任务。一旦人类的认知状态恢复到较低的认知负荷基线，主智能体便释放队列中的智能体消息。我们的初步结果表明，利用实时信号处理、大型语言模型（LLMs）和物理机器人实体来创建认知感知、非侵入式多智能体系统是可行的。

## Abstract
While non-verbal behaviors and expressive movements are essential for natural human-robot interaction, existing methods often overlook a crucial element: the human's internal cognitive state. Frequently, proactive multi-agent systems can interrupt humans at inopportune moments, leading to cognitive overload and decreased task performance. This paper introduces a framework for generating "cognitively aligned" multi-agent interactions, enhancing the ability of robotic systems to contextually defer communications to the user of an agent system during moments of high human mental workload and engagement. We present the design and implementation of a closed-loop architecture that explores the interplay between autonomous task execution and real-time neurophysiological focus. Using a consumer-grade Brain-Computer Interface (BCI), our approach continuously monitors Electroencephalography (EEG) spectral band powers while a human performs an engagement-inducing task. We propose an engagement-driven pipeline where an HTTP-based signaling mechanism places a primary agent's sensory inputs and audio outputs into a holding state upon detecting high engagement. This allows secondary agents to seamlessly process complex, delegated tasks in the background. Once the human's cognitive state returns to a lower cognitive load baseline, the primary agent releases the queued agent message. Our preliminary results demonstrate the feasibility of leveraging real-time signal processing, Large Language Models (LLMs), and physical robotic embodiments to create cognitively-aware, non-intrusive multi-agent systems.

## 2. 论文提出的方法论

### 2.1 核心思想
建立一个闭环架构，通过消费级脑机接口实时监测人类认知负荷（以投入度指数表示），当检测到高投入时，主代理暂停自身交互输出，由后台副代理继续处理任务，待用户认知负荷降低后再释放队列消息，实现多代理系统对人类认知状态的动态适应。

### 2.2 关键技术细节
- **脑机接口与认知负荷量化**：使用Muse头带采集EEG（4/5通道，256 Hz），通过快速傅里叶变换提取δ(1-4 Hz)、θ(4-8 Hz)、α(8-13 Hz)、β(13-30 Hz)、γ(30-45 Hz)频带功率。计算投入指数 R_t = P(β_t) / [P(α_t) + P(θ_t)]。经指数移动平均平滑得到最终投入分数 E_t = α * R_t + (1-α) * E_{t-1}（α设为0.15）。
- **认知对齐流水线**：当投入分数超过基线阈值的1.5倍并持续60秒后，2048游戏引擎发送HTTP POST请求("/focus?state=on")到主机，主代理暂停音频输入输出，LED变为红色表示保持状态。副代理后台继续处理任务。当投入分数下降，发送"/focus?state=off"请求，主代理恢复交互。
- **语义意图-中断分类器**：为保证紧急消息不被延误，引入本地量化LLM对消息重要性进行五级分类（0.0~1.0）。中断决策公式：I_t = I(W_imp > λ * E_t)，其中λ为中断厌恶常数，W_imp为消息重要性。若消息重要性超过当前投入度乘以常数，则允许中断。
- **代理生态系统**：主代理（Nadine）是带LED和扬声器的物理恐龙机器人，使用Google Gemini 3.1-flash-live-preview API进行实时语音交互；副代理（Red Claw、Blue Claw）是柔软龙虾机器人，使用Google Agent Development Kit (ADK)处理后台任务（如查询天气、机票价格）。副代理通过BLE和内部HTTPS协议通信。

1. 用户通过言语发出复杂指令（如“订去旧金山的航班”）。
2. 主代理解析意图，调用工具将任务委托给副代理，同时开始监测EEG。

4. 当EEG投入指数超过基线阈值1.5倍并持续60秒，游戏引擎向主机发送focus=on信号。

6. 副代理后台完成任务并合成消息，通过语义中断分类器判断重要性。

8. 用户完成游戏或认知负荷自然下降，发送focus=off信号，主代理恢复交互，释放队列消息。

## 9. 未来工作方向

- **动态自适应阈值**：当前使用固定倍数（1.5x基线）和60秒锁定窗口，未来可引入贝叶斯在线学习或强化学习，根据用户历史行为实时调整参数，减少过早释放与延迟响应。
- **多模态认知状态融合**：除EEG外，可结合眼动追踪、心率变异性（HRV）、皮肤电导（EDA）等外围生理信号，构建更稳健的认知负荷与情绪状态预测模型，区分“流畅专注”与“认知过载”。
- **高维EEG-LLM对齐**：将EEG原始时间序列或频带特征映射到LLM的嵌入空间，使代理不仅能感知“是否专注”，还能理解用户当前的思维主题或情绪基调（如困惑、兴奋），实现更深层次的认知对齐。
- **多用户与群体认知对齐**：当前仅支持单用户单代理，未来可扩展至多用户场景（如协作任务），多个BCI信号融合形成“群体认知状态”，由主代理协调多个副代理的交互策略。
- **纵向用户适应研究**：设计为期数周的用户日志实验，评估用户对系统行为的学习效应（如习惯保持红色LED段），以及长期使用对任务绩效和主观满意度的影响。
- **跨任务域泛化验证**：在写作、编程、驾驶模拟器等更真实的高认知负荷任务中测试系统，验证认知对齐框架是否具有任务独立性。
- **硬件集成与商业可行性**：将BCI模块微型化嵌入可穿戴设备（如智能眼镜、耳机），或与消费级机器人集成，降低部署门槛，实现日常化使用。
